SUNDMRG.jl

SUNDMRG.jl is a DMRG implementation with full SU(N) symmetry.

This documentation starts with the basic DMRG workflow, then covers package usage, example scripts, the representation-theoretic background used by the package, and the public API reference.

Reading Path

Start with DMRG Overview for the general algorithmic ideas. Use Usage for the first runnable calls and common options. The Examples page explains the scripts in the repository's examples/ directory. Then read SU(Nc) Representation Theory for the symmetry notation used in SU(Nc)-adapted calculations. Continue to Examples of SU(Nc) Representations for concrete labels and small tensor-product decompositions. The short Representation Labels in SUNDMRG.jl note records the row-length convention used when labels appear in package-facing contexts. Use API Reference when you need signatures and return fields.
